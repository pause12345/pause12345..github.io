FGA教程

2020年9月20日
| fgo-us

FGA是安卓专用的FGO辅助工具，类似手势模拟点击操作，不修改游戏数据，操作完全在FGO原版客户端上进行。

看起来问题不大，也不能说完全没风险，封号风险自负。

本教程针对英文版FGA。

一、下载安装

Google Play商店下载： https://play.google.com/store/apps/details?id=com.mathewsachin.fategrandautomata

分流下载

可以通过 安卓模拟器 BlueStacks运行FGA，必须在模拟器设置-引擎-图形引擎改为相容性（进阶模式），升级模拟器到最新版。

国产安卓各种改版系统可能会导致FGA不能正常使用，尽量用港行等海外原版安卓手机。

刘海屏手机有兼容问题，可以尝试主界面-More options-Advanced-启用ignore notch calculation。

二、初始配置

1、FGA主界面

  • Script Mode 保持默认的 Battle 战斗模式。
  • Refill 设置苹果消耗数量，看自己需要，Repetitions默认0就是不消耗。
  • Auto Skill 最重要的地方，配置战斗进入后怎么操作，不配置就是默认平A以及NP满了是否放宝具，后文会单独讲解，先跳过。
  • More options 见下文。

2、主界面More options

1）Game Server

默认NA就是美服，其他分别为JP 日服、CN B服、TW 台服、KR 韩服。

2）Battle

  • Skill Confirmation 技能确认，影响战斗节奏，不启用，FGO游戏里面也要设置成一样。
  • Story Skip 故事情节跳过，启用，自动战斗看啥剧情。。
  • Enable Withdraw 战败重试，默认不启用。
  • Stop on CE drop 掉落礼装停止战斗，通常不启用。
  • Stop on CE GET 第一次完成任务奖励礼装停止战斗，通常不启用。
  • Screenshot Drops 掉落截图，通常不启用。
  • Cast Noble Phantasm 释放宝具，默认不启用，可以改成spam 满了就用 或 danger 危险敌人或从者在场时释放宝具。
  • Auto choose target 自动选择攻击目标，会把危险敌人和从者设置为优先攻击目标。如果在战斗脚本设置了每回合具体攻击目标，这里需要设置为不启用。
  • Boost Item 战斗前使用的道具，可以根据情况选择不启用、跳过、道具1、道具2、道具3。如果要使用，确保FGO游戏里面设置不需要确认使用战斗道具。
  • Skill Command TextBox 编辑Auto Skill命令时候只输入文本命令，新手切勿启用。

3）Friend Gacha

Friend Points Summons Only 只进行友情点召唤，默认启用。

关闭的话，可以自动抽氪金池子，抽光所有石头为止。

4）Gift Box 礼物盒

Max Gold ember set size 礼物盒最大金狗粮数量，默认1。

超过1的金狗粮保留，其他的自动勾选。

5）Adavnced 高级设置

略

三、配置Auto Skill

1、建立新脚本

点击右下角+ 添加新的脚本。

2、脚本主界面

  • Name 给脚本取个自己看得懂的名字。

  • Command 模拟战斗界面设置攻击技能宝具释放等命令，想象实际战斗怎么点击的，多试试就知道了。进一步介绍见后文。

  • Notes 脚本注释，空着就行。

  • Party 队伍设置，根据实际情况设置，不设置就是当前队伍。

  • Card Priority 卡片优先次序，默认即可（WA, WB, WQ, A, B, Q, RA, RQ, RB 意思是 weak arts->weak buster->weak buster->arts->buster->quick->resist arts->resist buster->resist quick）。

  • Support 助战设置，见下文。

3、Support 助战设置

  • Class 职介，比如Caster术阶。
  • Selection Mode 选择模式，默认Preferred即可，会根据设定策略选择助战。
  • Fallback Selection Mode 后备选择模式，默认Manual，即玩家手动选择助战。预设刷新五次好友列表都没有找到合适助战时进入此模式。
  • Preferred Servants 偏爱从者，比如Skadi 即CBA，列表里面的几个自带的从者均支持各种灵基突破头像图标，这里可以使用自定义图标扩展。支持多选，按顺序匹配。
  • Max-asecnded 最大灵基突破，通常启用。
  • Skill Levels 最大技能，推荐10/10/10，CBA起码要13技能MAX。
  • Preferred CE 偏爱礼装，这里可以使用自定义图标扩展。支持多选，按顺序匹配。
  • Max-limit broken 礼装满破，除了活动刚开始，一般都启用。
  • Friends Only 只找朋友助战，通常启用，避免战斗结束卡在添加好友的界面。

比如这个就是闪闪祭2020的活动礼装图标

另外，也可以通过自带的Support Image Marker模式来抓取自定义图标：

  • 主界面Script Mode设置由默认的Battle 改为Support Image Marker模式。
  • 进入助战选择界面或者好友一览界面，把目标好友保持在最上方。
  • 点击左下角的▶️开始抓取图标，选择抓取从者或者礼装图标，取个名字保存。
  • 要注意从者需要抓取各个灵基突破以及灵衣的图标。

5、Command 战斗命令进阶

要启用上面提到的Skill Command TextBox直接编写命令。

1）战斗服换人

x - activates Order Change 
Starting Member Position - 1  2  3 
Sub-member position - 1  2  3 
e.g.  
Skill_Command = "x13" 
Exchange starting member 1 with sub-member 3

和普通命令混合使用示例

e.g. Skill_Command = "bce,0,f3hi,#,j2d,#,4,x13a1g3"

2）自定义攻击目标

t - informs script of targeting
Position of enemy - 1 2 3

e.g.
Skill_Command = "t1at35"
Target far left enemy for using Servant 1 skill 1, then target far right enemy for using Servant 1 NP

把上面提到的Auto choose target 自动选择攻击目标关闭。

3）宝具前使用命令卡攻击

Skill_Command = "n145"
Use 1 regular Command Card according to the priority from Battle_CardPriority, then use the NPs of the first and second Servants

Skill_Command = "n26"
Use 2 regular Command Cards, then use the NP of the third Servant

四、使用FGA

1、运行FGA，配置完成后，主界面点击 TOGGLE SERVICE，必须给予需要的权限。

安卓系统设置accessibility（辅助功能/无障碍功能），这里要开启FGA ON。

2、运行FGO，停留在以下位置

  • 菜单界面，将目标任务作为界面上的第一项（右上角）
  • 战斗界面，未做任何技能释放等操作
  • 战斗结果界面
  • 助战选择界面

3、点击左下角的▶️开始运行FGA自动战斗脚本，运行中点击⏹停止。

4、注意手机设置不要自动锁屏休眠了，需要保持开启状态。

5、补充说明主界面Script Mode设置选项

  • Battle 自动战斗，默认。
  • Lottery 无限池抽奖，抽光自动重置池子。
  • Friend Gacha 抽友情池，默认只抽友情池。
  • Support Image Maker 抓取助战图片。
  • Gift Box 礼物盒清狗粮，默认保留1以上的金狗粮，可以改。
